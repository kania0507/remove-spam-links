Plugin removes spam links from comment area. 

== Description ==

Remove spam link plugin checks comment text before display it and prevent your site from publishing malicious links. 
Links in comments text area are not visible. 

== Installation ==

1. Download the plugin and unzip it.
2. Upload the folder remove-spam-links/ to your /wp-content/plugins/ folder.
3. Activate the plugin from your WordPress admin panel.
4. Installation finished.
